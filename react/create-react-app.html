# create-react-app

# 搭建开发环境

1. 必须安装nodejs 注意：安装nodejs稳定版本 没安装的可以参考通过nvm来安装管理node (opens new window)

2. 安装cnpm用cnpm替代npm

  • 地址：http://npm.taobao.org/ (opens new window)
  • npm install -g cnpm --registry=https://registry.npm.taobao.org

3. 用yarn替代npm yarn的安装：
第一种方法：参考官方文档https://yarn.bootcss.com/ (opens new window)
第二种方法：cnpm install -g yarn 或者 npm install -g yarn

# 安装脚手架

npm install -g create-react-app / cnpm install -g create-react-app（推荐）

  • 注意需要添加 -g 参数进行全局安装以及权限的问题。

# 创建项目

create-react-app reactdemo

  • 安装yarn之后会默认用yarn来创建项目（个人感觉下载速度更快）

# 运行项目

cd reactdemo

npm start/yarn start

npm run build/yarn build

# 用npx创建项目

# 安装脚手架工具并创建项目

npx create-react-app reactdemo

# npx介绍

npm v5.2.0引入的一条命令（npx），引入这个命令的目的是为了提升开发者使用包内提供的命令行工具的体验详情 (opens new window)

npx create-react-app reactdemo这条命令会临时安装 create-react-app包，命令完成后create-react-app 会删掉，不会出现在 global 中。下次再执行，还是会重新临时安装。 npx 会帮你执行依赖包里的二进制文件。
